# Next.js 2 with Material-UI Example

## Init node project
```
npm init -y
npm i -S next react react-dom material-ui react-tap-event-plugin
```

## Add script to `package.json`
```json
"scripts": {
  "dev": "next",
  "build": "next build",
  "start": "next start",
  "test": "echo \"Error: no test specified\" && exit 1"
}
```

## Create index page
```
mkdir pages && cd pages
touch index.js
```
